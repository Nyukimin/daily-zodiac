# Daily Zodiac 画面仕様

最終更新: 2026-02-11

## 概要

モバイルファーストの日次星座占いサイト。SNS からの流入を想定し、星座の切り替えを最上部に配置。

---

## ページ構成

| 画面 | パス | 説明 |
|------|------|------|
| HOME（全体運） | `/` | 12 星座の「全体運」を表示。RANAI キービジュアルを使用 |
| 星座ページ | `/{sign}/` | 各星座の個別運勢。カラーキービジュアルを使用 |

---

## レイアウト（上から順）

### 1. 星座切り替えタブ

- **配置**: 最上部（メインコンテンツの先頭）
- **理由**: SNS 流入時、即座に星座を切り替え可能にするため
- **タップ領域**: `min-h-[44px] min-w-[44px]` 以上（Apple HIG 準拠）
- **レイアウト**: 横スクロール（`overflow-x-auto`）、チップ形式 `rounded-lg bg-white/5`
- **内容**:
  - 全体運（HOME へのリンク）
  - 牡羊座、牡牛座、双子座、蟹座、獅子座、乙女座、天秤座、蠍座、射手座、山羊座、水瓶座、魚座
- **表示**:
  - 全体運選択時: 「全体運」のみ bold + 白 + bg-white/15
  - 星座選択時: 該当星座のみ bold + 白 + bg-white/15
- **スタイル**: `text-xs`、`px-4 py-2`、`whitespace-nowrap`

---

### 2. 占い結果（キービジュアル）

- **ラベル**: なし（上部の「占い結果」テキストは表示しない）
- **余白**: コンテンツ領域に `pt-3 pb-3` を付与し、イラストがボックス最上部まで伸びないようにする
- **高さ**: `min-h-[160px]`、内容に応じて可変。縦スクロールで全て閲覧可能
- **背景**: `bg-zinc-900/80`
- **レイアウト**: 左にテキスト、右にイラスト（flex）。左右比率 テキスト `flex-[0.9]`、イラスト `flex-[1.3]`

#### 左エリア（テキスト）

| 行 | 内容 | 条件 |
|----|------|------|
| 1 | `{{ data.date }}` | 日付のみ（例: 2026-02-11） |
| 2 | 「今日の全体運」 | HOME の場合 |
| 2 | 「今日の{{ data.sign_ja }}」 | 星座ページの場合（例: 今日の牡羊座） |
| 3 | 要約テキスト | **全文表示**（省略なし） |

- **スタイル**: 行1 `text-[10px]`、行2 `text-base font-semibold`、行3 `text-xs`

#### 右エリア（イラスト）

- **イラストエリア高さ**: `h-[200px]` 固定
- **画像**: 高さ指定なし。`w-[130%]`、`object-contain object-center`
- **配置**: 画像の縦中心と占い結果ボックスの縦中心を揃える（親の `items-center` で実現）
- **オーバーフロー**: 親の `overflow-hidden` ではみ出しをクリップ
- **グラデーション**: 左端（zinc-900 `#18181b`）から右へなだらかに透明化
  - 0% → 30% → 55% → 78% → 100%（transparent）
- **キービジュアル画像**:
  - HOME: `images/RANAI/RANAI_*.png`（日付でローテーション）
  - 星座: `images/カラー/{色名}/*.png`（星座・日付で選択）

---

### 3. 広告枠

- **ラベル**: 「スポンサーリンク」
- **枠**: 300×48px、`max-w-[300px] h-10`
- **スタイル**: `bg-zinc-950/50 ring-1 ring-white/10`

---

### 4. アドバイス（結果コンテンツ）

高さは内容に応じて可変。長文は折り返し表示。縦スクロールで全て閲覧可能。

#### 4.1 ラベル

- **ラベル**: 「アドバイス」（`text-[10px] text-zinc-200/80`、上部に表示）

#### 4.2 本文

- **本文**: `{{ data.advice }}`
- **スタイル**: `text-sm font-semibold leading-relaxed`

#### 4.3 注意書き

- **本文**: 「※占いは娯楽です。医療・法律・投資などの判断材料としては使いません。」
- **配置**: 最下部、`shrink-0`

## レスポンシブ・レイアウト

- **ベース**: モバイル縦画面、スクロール可能なシングルカラム
- **コンテナ**: `max-w-5xl px-4 py-4`
- **main**: `flex flex-col min-h-screen`（固定ビューポート依存なし、縦スクロールで全コンテンツ閲覧可能）

---

## テンプレート構成

```
base.html
├── 星座切り替えタブ (nav)
├── 占い結果（キービジュアル） (section)
├── 広告枠 (aside)
└── アドバイス（結果コンテンツ） (section)
    └── {% block content %} → hero_content.html
```

- **index.html**: base を継承、HOME 用（`is_home=True`）
- **sign.html**: base を継承、星座ページ用（`is_home=False`）
- **hero_content.html**: アドバイス（ラベル付き）・注意書き

---

## データ変数

| 変数 | 型 | 説明 |
|------|-----|------|
| `data.date` | str | 日付 YYYY-MM-DD |
| `data.sign` | str | 星座スラッグ（例: aries） |
| `data.sign_ja` | str | 星座名（例: 牡羊座） |
| `data.summary` | str | 要約文 |
| `data.advice` | str | アドバイス文 |
| `signs` | list[(str,str)] | (スラッグ, 日本語名) のリスト |
| `base_href` | str | ベース URL（例: /daily-zodiac/） |
| `is_home` | bool | HOME か否か |
