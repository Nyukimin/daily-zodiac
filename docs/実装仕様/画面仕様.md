# Daily Zodiac 画面仕様

最終更新: 2026-02-11

## 概要

モバイルファーストの日次星座占いサイト。SNS からの流入を想定し、星座の切り替えを最上部に配置。

---

## ページ構成

| 画面 | パス | 説明 |
|------|------|------|
| HOME（全体運） | `/` | 12 星座の「全体運」を表示。RANAI キービジュアルを使用 |
| 星座ページ | `/{sign}/` | 各星座の個別運勢。カラーキービジュアルを使用 |

---

## レイアウト（上から順）

### 1. 星座切り替えタブ

- **配置**: 最上部（メインコンテンツの先頭）
- **理由**: SNS 流入時、即座に星座を切り替え可能にするため
- **内容**:
  - 全体運（HOME へのリンク）
  - 牡羊座、牡牛座、双子座、蟹座、獅子座、乙女座、天秤座、蠍座、射手座、山羊座、水瓶座、魚座
- **表示**:
  - 全体運選択時: 「全体運」のみ bold + 白
  - 星座選択時: 該当星座のみ bold + 白
- **スタイル**: `text-[10px] text-zinc-200/80`、flex-wrap、gap

---

### 2. キービジュアル

- **高さ**: `h-[28vh] min-h-[120px]`
- **背景**: `bg-zinc-900/80`
- **レイアウト**: 左にテキスト、右にイラスト（flex）

#### 左エリア（テキスト）

| 行 | 内容 | 条件 |
|----|------|------|
| 1 | `{{ data.date }}` | 日付のみ（例: 2026-02-11） |
| 2 | 「今日の全体運」 | HOME の場合 |
| 2 | 「今日の{{ data.sign_ja }}」 | 星座ページの場合（例: 今日の牡羊座） |
| 3 | 要約テキスト | 45 文字超は「…」で省略 |

- **スタイル**: 行1 `text-[10px]`、行2 `text-base font-semibold`、行3 `text-xs`

#### 右エリア（イラスト）

- **画像**: `h-[130%] max-w-none`、`object-contain object-right object-bottom`
- **グラデーション**: 左端（zinc-900 `#18181b`）から右へなだらかに透明化
  - 0% → 30% → 55% → 78% → 100%（transparent）
- **キービジュアル画像**:
  - HOME: `images/RANAI/RANAI_*.png`（日付でローテーション）
  - 星座: `images/カラー/{色名}/*.png`（星座・日付で選択）

---

### 3. 広告枠

- **ラベル**: 「スポンサーリンク」
- **枠**: 300×48px、`max-w-[300px] h-10`
- **スタイル**: `bg-zinc-950/50 ring-1 ring-white/10`

---

### 4. 結果コンテンツ

#### 4.1 要約

- **ラベル**: 「要約」
- **本文**: `{{ data.summary }}`（省略なし）

#### 4.2 選択肢

- **表示**: 最大 3 件、2〜3 カラムグリッド
- **各カード**: `{{ data.sign }}/` への同一リンク（現状）
- **スタイル**: `rounded-xl bg-white/10 ring-1 ring-white/15`

#### 4.3 次の一歩

- **左**: 「次の一歩」ラベル + `{{ data.next_step }}`
- **右**: 「もう一回みる」ボタン → HOME へ
- **スタイル**: 白ボタン `bg-white text-zinc-950`

#### 4.4 注意書き

- **本文**: 「※占いは娯楽です。医療・法律・投資などの判断材料としては使いません。」

## レスポンシブ

- **ベース**: モバイル縦画面
- **コンテナ**: `max-w-5xl px-4 py-4`
- **選択肢**: `grid-cols-2 sm:grid-cols-3`

---

## テンプレート構成

```
base.html
├── 星座切り替えタブ (nav)
├── キービジュアル (section)
├── 広告枠 (aside)
└── 結果コンテンツ (section)
    └── {% block content %} → hero_content.html
```

- **index.html**: base を継承、HOME 用（`is_home=True`）
- **sign.html**: base を継承、星座ページ用（`is_home=False`）
- **hero_content.html**: 要約・選択肢・次の一歩・注意書き

---

## データ変数

| 変数 | 型 | 説明 |
|------|-----|------|
| `data.date` | str | 日付 YYYY-MM-DD |
| `data.sign` | str | 星座スラッグ（例: aries） |
| `data.sign_ja` | str | 星座名（例: 牡羊座） |
| `data.summary` | str | 要約文 |
| `data.choices` | list | 選択肢（最大 3 件） |
| `data.next_step` | str | 次の一歩 |
| `signs` | list[(str,str)] | (スラッグ, 日本語名) のリスト |
| `base_href` | str | ベース URL（例: /daily-zodiac/） |
| `is_home` | bool | HOME か否か |
